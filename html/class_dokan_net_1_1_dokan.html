inlinestatic

Mount a new Dokan Volume. It is mandatory to have called DokanInit previously to use this API. This function returns directly on device mount or on failure. WaitForFileSystemClosed can be used to wait until the device is unmount.

Parameters
operationsInstance of IDokanOperations that will be called for each request made by the kernel.
mountPointMount point. Can be M:\ (drive letter) or C:\mount\dokan (path in NTFS).
loggerILogger that will log all DokanNet debug informations
Exceptions
DokanExceptionIf the mount fails.
Returns
Dokan mount instance context that can be used for related instance calls like IsFileSystemRunning

static void DokanNet.Dokan.Init ( )
inlinestatic

Initialize all required Dokan internal resources.

This needs to be called only once before trying to use Mount or CreateFileSystem for the first time. Otherwise both will fail and raise an exception.
